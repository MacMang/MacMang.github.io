08.canvas

Canvas

Canvas元素用来支持在网页上绘制图形图像,并在Canvas元素上实现动画.

Canvas元素

<canvas> 标签只有两个属性—— width和height,它们都是可选的.我们可以通过JS的方式给它动态设置宽高

  <canvas id="canvas" width="150" height="150"></canvas>

2D渲染上下文

canvas元素只是为我们创建了一个可以画画的地方,而我们真正要画画,还是需要在JS中使用canvas提供的2D渲染上下文.<canvas> 元素有⼀一个叫做 getContext() 的⽅方法，这个⽅方法是⽤用来获得渲染 上下⽂文和它的绘画功能。getContext()只有⼀一个参数，上下⽂文的格式。

var canvas = document.getElementById('canvas');
var ctx = canvas.getContext('2d');

栅格和坐标系统

canvas元素默认被⽹网格所覆盖,通常来说⽹网格中的⼀一个单元相当于canvas元素中的 ⼀一像素。栅格的左上角为原点（0,0,向右移动时，x坐标值会增加；向下移动时，y坐标值会增加

绘制图形

矩形

canvas提供了了三种⽅方法绘制矩形

  1. fillRect(x, y, width, height) 绘制⼀一个填充的矩形
  2. strokeRect(x, y, width, height) 绘制⼀一个矩形的边框
  3. clearRect(x, y, width, height) 清除指定矩形区域，让清除部分完全透明
    var canvas = document.getElementById('canvas');
    if (canvas.getContext) {
        var ctx = canvas.getContext('2d');
        ctx.fillRect(25,25,100,100);
        ctx.clearRect(45,45,60,60);
        ctx.strokeRect(50,50,50,50);
    }

绘制路路径

图形的基本元素是路路径。路路径是通过不不同颜⾊色和宽度的线段或曲线相连形成的不不同 形状的点的集合。⼀一个路路径，甚⾄至⼀一个⼦子路路径，都是闭合的。使⽤用路路径绘制图形需 要⼀一些额外的步骤。

  1. ⾸首先，你需要创建路路径起始点。
  2. 然后你使⽤用画图命令去画出路路径。
  3. 之后你把路路径封闭。
  4. 一旦路径⽣生成，你就能通过描边或填充路路径区域来渲染图形。

以下是所要⽤用到的函数:

  • beginPath() 新建⼀一条路路径，⽣生成之后，图形绘制命令被指向到路路径上⽣生成路路径。 closePath() 闭合路路径之后图形绘制命令⼜又重新指向到上下⽂文中。
  • stroke() 通过线条来绘制图形轮廓。
  • fill() 通过填充路路径的内容区域⽣生成实⼼心的图形。

⽣成路路径的第一步叫做beginPath()。本质上，路径是由很多子路径构成，这些子路 径都是在⼀个列列表中，所有的⼦路径(线、弧形、等等)构成图形。⽽而每次这个方法调⽤之后，列表清空重置，然后我们就可以重新绘制新的图形 第⼆步就是调⽤用函数指定绘制路路径，本⽂稍后我们就能看到了了。 第三，就是闭合路径closePath(),不是必需的。这个⽅法会通过绘制⼀条从当前点到 开始点的直线来闭合图形。如果图形是已经闭合了的，即当前点为开始点，该函数 什么也不不做。

绘制⼀一个三⻆角形

var canvas = document.getElementById('canvas');
if (canvas.getContext) {
    var ctx = canvas.getContext('2d');
    ctx.beginPath();
    ctx.moveTo(75, 50);
    ctx.lineTo(100, 75);
    ctx.lineTo(100, 25);
    ctx.fill();
}

圆弧

绘制圆弧或者圆，我们使用arc()⽅方法

arc(x, y, radius, startAngle, endAngle, anticlockwise) 画⼀一个以(x,y)为圆⼼的以radius为半径的圆弧(圆)，从startAngle开始到 endAngle结束，按照anticlockwise给定的⽅向(默认为顺时针)来⽣生成。 arcTo(x1, y1, x2, y2, radius) 根据给定的控制点和半径画一段圆弧，再以直线连接两 个控制点。

这⾥详细介绍一下arc方法，该⽅法有六个参数:x,y为绘制圆弧所在圆上的圆⼼心坐 标。radius为半径。startAngle以及endAngle参数⽤用弧度定义了了开始以及结束的弧 度。这些都是以x轴为基准。参数anticlockwise为⼀一个布尔值。为true时，是逆时针 ⽅方向，否则顺时针⽅方向

 注意:arc()函数中的⻆角度单位是弧度， 不不是度数。⻆角度与弧度的js表达式: 
 radians=(Math.PI/180)*degrees。

绘制12个不同的⻆度以及填充的圆弧。

 if (canvas.getContext) {
        var ctx = canvas.getContext('2d');
        for (var i = 0; i < 4; i++) {
            for (var j = 0; j < 3; j++) {
                ctx.beginPath();
                var x = 25 + j * 50;
                // x 坐标值 // y 坐标值 // 圆弧半径 0; // 开始点
                var y = 25 + i * 50;
                var radius = 20;
                var startAngle = 0;
                var endAngle = Math.PI + (Math.PI * j) / 2; // 结束点
                var anticlockwise = i % 2 == 0 ? false : true; // 顺时针或逆时针
                ctx.arc(x, y, radius, startAngle, endAngle, anticlockwise);
                if (i > 1) {
                    ctx.fill();
                } else {
                    ctx.stroke();
                }
            }
        }
    }

色彩 Colors

fillStyle = color 设置图形的填充颜色。 strokeStyle = color 设置图形轮廓的颜色。 color 可以是表示 CSS 颜⾊值的字符串，渐变对象或者图案对象。我们迟些再回头 探讨渐变和图案对象。默认情况下，线条和填充颜色都是⿊色(CSS 颜⾊值 #000000)。

注意: ⼀旦您设置了 strokeStyle 或者 fillStyle 的值，那么这个新值就会成为新绘制 的图形的默认值。如果你要给每个图形上不同的颜⾊色，你需要重新设置 fillStyle 或 strokeStyle 的值

function findPos(obj) {
    var curleft = 0,
        curtop = 0;
    if (obj.offsetParent) {
        do {
            curleft += obj.offsetLeft;
            curtop += obj.offsetTop;
        } while ((obj = obj.offsetParent));
        return {
            x: curleft,
            y: curtop
        };
    }
    return undefined;
}

function getColor(x, y, ctx) {
    var pixel = ctx.getImageData(x, y, 1, 1),
        data = pixel.data,
        rgb = 'rgb(' + data[0] + ',' + data[1] + ',' + data[2] + ')';
    colors.r = data[0];
    colors.g = data[1],
    colors.b = data[2];
    colors.a = 1;
    return rgb
}

变形 Transformations

状态的保存和恢复 Saving and restoring state

save 和 restore ⽅方法是⽤用来保存和恢复 canvas 状态的，都没有参 数。Canvas 的状态就是当前画⾯面应⽤用的所有样式和变形的⼀一个快照。 Canvas状态存储在栈中，每当save()方法被调⽤用后，当前的状态就被推送到栈中保存.你可以调⽤任意多次 save 方法。每⼀次调⽤restore ⽅法，上一个保存的状态就从栈中弹出，所有设定都恢复

save 和 restore 的应⽤用例例⼦子

我们尝试⽤用这个连续矩形的例例⼦子来描述 canvas 的状态栈是如何⼯工作的。 第⼀步是用默认设置画一个⼤四⽅方形，然后保存⼀下状态。改变填充颜色画第二个 ⼩一点的蓝⾊色四⽅方形，然后再保存一下状态。再次改变填充颜色绘制更更⼩一点的半 透明的白⾊四⽅形。

到目前为⽌所做的动作和前⾯章节的都很类似。不过⼀旦我们调⽤ restore，状态栈中最后的状态会弹出，并恢复所有设置。如果不是之前用 save 保存了了状态，那么我 们就需要⼿动改变设置来回到前⼀个状态，这个对于两三个属性的时候还是适⽤ 的,一旦多了了，我们的代码将会猛涨。

当第⼆次调⽤用 restore 时，已经恢复到最初的状态，因此最后是再一次绘制出⼀个 ⿊色的四方形。


var ctx = document.getElementById('canvas').getContext('2d');
ctx.fillRect(0, 0, 150, 150); // 使⽤用默认设置绘制⼀个矩形
ctx.save(); // 保存默认状态
ctx.fillStyle = '#09F' // 在原有配置基础上对颜⾊做改变
ctx.fillRect(15, 15, 120, 120);  // 使⽤用新的设置绘制⼀个矩形
ctx.save();  // 保存当前状态
ctx.fillStyle = '#FFF'; // 再次改变颜色配置

ctx.fillRect(30, 30, 90, 90); //使⽤用新的配置绘制⼀个矩形
ctx.restore();  //重新加载之前的颜⾊状态
ctx.fillRect(45, 45, 60, 60);// 使⽤上一次的配置绘制⼀个矩形

ctx.fillRect(60, 60, 30, 30);//使⽤用加载的配置绘制⼀个矩形

移动 Translating

我们先介绍 translate 方法，它用来移动 canvas 和它的原点到一个不同的位置。

ranslate(x, y) translate ⽅法接受两个参数。x 是左右偏移量，y 是上下偏移量，如 下图所示。 在做变形之前先保存状态是⼀个良好的习惯。大多数情况下，调⽤ restore ⽅法⽐⼿动恢复原先的状态要简单得多。⼜如果你是在一循环中做位移 但没有保存和恢复 canvas 的状态，很可能到最后会发现怎么有些东西不见了了，那 是因为它很可能已经超出 canvas 范围以外了

translate 的例子

匀速直行

// 创建一个球
function Ball(radius,color){
    var radius = radius || 40;
    var color = color || "lightblue";
    this.x = 0;
    this.y = 0;
    this.radius = radius;
    this.color = color;
}
// 为Ball的原型对象设置一个draw方法,每次发生变化就调用draw方法绘图
Ball.prototype.draw = function(context){
    context.save();
    context.translate(this.x,this.y);
    context.fillStyle = this.color;
    context.strokeStyle = this.color;
    context.beginPath();
    context.arc(0,0,this.radius,0,Math.PI*2,false);
    context.closePath();
    context.fill();
    context.stroke();
    context.restore();
}
